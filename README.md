## პროექტის გაშვება
პროექტი `sail` ით მუშაობს, სეილით გარემოს გამართვისა და `migrate:fresh --seed` ის შემდგომ
გაუშვით `sail artisan app:import-countries` ქვეყნების წამოსაღებად



## ქართული ლოკალიზაცია

`Accept-Language` Header-ი `ka` მნიშვნელობით
